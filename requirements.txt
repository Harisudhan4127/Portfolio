PROJECTS
Wearable Inhaler Device with AI for Real-Time Asthma Risk Assessment
This is my final year project focused on developing a smart wearable device that monitors environmental and physiological parameters to predict asthma risk in real time.

The system uses ESP8266, sensors, and AI-based analysis to provide alerts and display health insights on a TFT screen with a smartwatch-style interface. The project aims to improve preventive healthcare using intelligent monitoring systems.

Bluetooth-Based Water Flow Meter
Developed a smart water monitoring system using Arduino and Bluetooth connectivity. The system calculates water flow and displays real-time and 1-minute average data on both an LCD and a mobile application built using MIT App Inventor.
Gesture-Based Virtual Mouse using Computer Vision
Designed a system using OpenCV and MediaPipe to control the computer cursor using hand gestures. This project explores human-computer interaction and aims to extend functionality with voice assistant integration.
MAC Address-Based Attendance System
Created a web-based attendance tracking system that automatically marks attendance by detecting and matching students' MAC addresses over a network.
Embedded System Projects & Prototyping
